Detailed Description

The AV1 RTP aggregation header byte (AOMedia AV1 RTP §4.4).

Bit layout (network byte order, bit 0 is the MSB): 

 0 1 2 3 4 5 6 7
+-+-+-+-+-+-+-+-+
|Z|Y| W |N|-|-|-|
+-+-+-+-+-+-+-+-+
